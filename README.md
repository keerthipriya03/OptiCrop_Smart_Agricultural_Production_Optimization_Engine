# OptiCrop 🌱

Machine Learning based crop recommendation system.

## Overview

OptiCrop predicts the most suitable crop based on soil and environmental conditions.

## Features

- Crop recommendation based on:
  - Nitrogen
  - Phosphorus
  - Potassium
  - Temperature
  - Humidity
  - pH
  - Rainfall

- Machine learning based prediction
- Flask web interface

## Technologies Used

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Python
- Flask

Machine Learning:
- Scikit-learn
- Pandas
- Joblib

## Run Locally

Clone repository:

```bash
git clone https://github.com/keerthipriya03/OptiCrop
````

Install dependencies:

```bash
pip install -r requirements.txt
```

Run:

```bash
python app.py
```

Open:

```
http://127.0.0.1:5000
```
